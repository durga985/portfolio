# Durga Portfolio (Next.js + Tailwind)

## Run locally
1. Install Node.js 18+.
2. In this folder:

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Customize
All content is in `data/profile.ts`.

### Add your links
Update each project's `links.github`, `links.demo`, `links.writeup` fields.

## Deploy
- **Vercel**: import the repo and deploy.
- **GitHub Pages**: possible, but Vercel is simplest for Next.js.

Resume PDF is served from `public/resume.pdf`.
